## 修改背景图

1. 在包**hexo-theme-icarus**中找到source/img文件夹，在其中添加背景图。
2. 然后在source/css/style.styl文件中添加自定义，并新建文件。

<!--more-->

```stylus
//style.styl
@import 'custom'
//custom.styl
body::before {
    content: '';
    position: fixed;
    z-index: -1;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    opacity: .01;
    transition: opacity 1s;
    background-image: url('../img/bg.png');
    background-position: center center;
    background-size: cover;
}
body.ready::before {
    opacity: 1;
}
```

3. 最后在source/js/main.js文件中添加代码

```stylus
    window.onload = () => {
        setTimeout(() => {
            document.querySelector('body').classList.add('ready')    
        }, 1000);
    }
```
